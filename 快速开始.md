# 🐍 贪吃蛇游戏 - 快速开始指南

## 🚀 立即运行

### 方法 1: 双击打开（最简单）
直接在浏览器中双击 `index.html` 文件即可开始游戏

### 方法 2: 本地服务器（推荐）
```bash
# 进入项目目录
cd tancise

# 启动简单HTTP服务器（Python）
python3 -m http.server 8080

# 然后在浏览器访问: http://localhost:8080
```

### 方法 3: 在线浏览（最方便）
如果项目已部署到 GitHub Pages，直接访问：
```
https://wangfugui1799.github.io/tancise/
```

**⚠️ 提示**: 如果还没有启用 GitHub Pages，可以这样操作：
1. 在 GitHub 仓库的 Settings → Pages
2. Source 选择 "main" 分支，目录选择 "/ (root)"
3. 点击 Save，等待几分钟即可访问

## 🎮 操作控制

| 功能 | 按键 | 说明 |
|------|------|------|
| **移动** | ↑↓←→ 或 WASD | 控制蛇的方向 |
| **暂停/继续** | 空格键 | 暂停或继续游戏 |
| **重新开始** | R 键 | 游戏结束后重新开始 |

## 📊 游戏特性

### ✅ 已实现功能
- **经典玩法**: 移动、吃食物、变长、不能撞墙撞自己
- **动态速度**: 分数越高，速度越快（每50分加速一次）
- **计分系统**: 实时得分和最高分记录
- **本地持久化**: 最高分保存在浏览器中
- **响应式设计**: 适配桌面和移动设备
- **视觉反馈**:
  - 蛇头带眼睛（跟随方向）
  - 食物闪烁动画
  - 得分动画效果
  - 游戏状态遮罩

### 🎨 美观界面
- 深色主题设计（护眼）
- 渐变色彩效果
- 圆角UI元素
- 平滑动画过渡

## 🔧 技术亮点

1. **面向对象设计**: 游戏封装在 `Game` 类中
2. **精确定时器**: 使用 `requestAnimationFrame` + 时间戳实现稳定游戏循环
3. **碰撞检测**: 墙壁碰撞 + 自身碰撞
4. **本地存储**: `localStorage` 持久化最高分
5. **音效系统**: Web Audio API 动态音效
6. **键盘防抖**: 防止方向键滚动页面

## 🐛 已知限制

## 💡 提示与技巧

1. **保持冷静**: 游戏越往后越快，保持冷静才能打破记录
2. **预判路径**: 提前规划蛇的移动路径
3. **利用空间**: 不要只在小范围内转圈，充分利用空间
4. **蛇身技巧**: 可以利用自己的身体围住食物（高手技巧）
